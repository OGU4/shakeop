# サーモンラン NEXT WAVE — ゲームルール詳細

本ドキュメントはサーモンランNW（Splatoon 3「サーモンラン NEXT WAVE」）のゲームルールを記録する。
本リポジトリのアプリ開発においてドメイン知識として参照するためのものである。

> **注意**: 本ドキュメントはアプリ開発に関連するルールを中心に記載しており、
> ゲーム内の全ルールを網羅するものではない。

## バイトの流れ

1. マッチング後バイトが始まると、各プレイヤーにバイト専用装備のセットが支給される。
2. シフトで決められたステージのスタート地点からゲームスタート。
3. WAVE 1 が始まり、成功条件を満たすと次の WAVE 2 がスタート。同様に WAVE 3 へ。
4. 最終 WAVE 3 をクリアすることでバイトは成功。リザルト画面へ。
5. クリア条件が満たされなかった場合「Works Over!」となりバイト失敗。
6. 仮に WAVE 1 で失敗した場合も次の WAVE が始まるといったことはなく、即リザルト画面へ。
7. リザルト画面では各種ポイントの入手、減算などが行われる。

## WAVE について

バイト1回のなかに WAVE（ウェーブ）と呼ばれる区切りが存在する。

### WAVE の構成

ひとつの WAVE は以下の3フェーズで構成されている：

| フェーズ | カウント | 説明 |
|---|---|---|
| 準備時間 | 10カウント | WAVE開始直後。そのWAVEで使用するメインウェポンが支給される |
| 本番 | 100カウント | シャケが出現し、金イクラの回収・納品を行うメインフェーズ |
| 休憩時間 | 短時間 | WAVEクリア時のみ発生。次のWAVEへの移行期間 |

### WAVE の種類

| 種別 | WAVE数 | 備考 |
|---|---|---|
| 通常のバイト | Wave 1 ~ 3 + Extra Wave | 計4種類 |
| バイトチームコンテスト | Wave 1 ~ 5 | 不定期開催。計5種類 |

### 準備時間・休憩時間の挙動

- 準備時間・休憩時間中にもキャラを操作でき、ナワバリを広げ足場を固めること等ができる。
- 水位の変化・空の暗転などの「特殊な状況」への環境変化が起こることもある。
- 特に水位の変化では納品場所であるコンテナの位置が変わることがある。

### 本番中の挙動

- 100カウントの本番中は、ステージごとに決められた地点からザコシャケ・オオモノシャケが定期的に現れる。
- バイトの難易度が上がるほど、シャケたちがより多く現れるようになる。

### WAVE 間の移行

- WAVE 終了時点でイクラコンテナ外にある金イクラは、手元にあるものを含めてすべて消滅する。
- WAVE 終了後の小休憩が終わると強制的にスタート地点へスーパージャンプする。
- 着地後はウキワ状態からでも復活し、次の WAVE の準備時間が始まる。
- リスポーン時間等でスーパージャンプが遅れる場合や、ウキワ状態から復活すると操作可能になるまで少し時間がかかる場合があるが、開始時間は変わらない。

## WAVE の成功・失敗条件

### 成功条件

- 既定のノルマ数以上の納品を達成していること。
- かつ、タイムアップまで生存状態のプレイヤーがいる（生存可能性がある）こと。
- クリア後の小休憩時間に流れ弾や水没で全滅するのはセーフ。

### 失敗条件

以下のいずれかを満たした場合、バイト失敗となる：

- 既定のノルマ数の納品に満たないまま制限時間を終えること。
- 100カウント中にプレイヤー全員がやられること（生存可能性がなくなること）。

## ノルマについて

- ひとつの WAVE ごとに回収すべき金イクラ運搬のノルマが設定されている。
- 「オンライン時の参加プレイヤーの称号や評価値の合計が高い」「WAVE が進む」ほどノルマは増える。

## シャケとイクラ、納品について

### イクラの入手

- ザコ・オオモノ問わずシャケにダメージを与えると、ダメージを与えたプレイヤーがイクラ（金イクラではない）を自動入手する。
- 止めを刺せなくとも一部入手できる。

### 金イクラの入手

- オオモノシャケなど特定のシャケを倒すとその場に金イクラを落とす。
- 水中などプレイヤーが侵入できない場所で倒した場合は、プレイヤーが獲得できる別地点に落ちることがある。

### 金イクラの運搬と納品

- 金イクラに接触することで回収し、プレイヤー1人につき1つまで保持できる。
- 保持状態でやられても金イクラは奪われることはない。
- 保持状態でイクラコンテナ前まで運搬し、生存状態であれば A ボタンを押すことで納品できる。
- 保持しただけの金イクラや出現した金イクラはノルマ数には一切カウントされない。
- **コンテナに納品した分のみがノルマとしてカウントされる。**
